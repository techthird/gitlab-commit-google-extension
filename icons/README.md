# 图标文件

请在此文件夹中放置以下尺寸的图标文件：

- `icon16.png` - 16x16像素
- `icon48.png` - 48x48像素  
- `icon128.png` - 128x128像素

图标可以是GitLab相关的图标，或者使用任何你喜欢的图标。

如果没有图标文件，扩展程序仍然可以正常工作，只是工具栏中不会显示图标。

## 快速创建图标

你可以使用在线工具创建图标，或者使用以下方法：

1. 使用在线图标生成器：https://www.favicon-generator.org/
2. 使用设计工具（如Figma、Photoshop）创建
3. 使用简单的GitLab logo作为图标
